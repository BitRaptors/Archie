# Archie Init

This command has been renamed. Use one of:

- **`/archie-scan`** — Fast architecture health check (1-3 min). Run often. Recommended starting point.
- **`/archie-deep-scan`** — Comprehensive architecture baseline (15-20 min). Run once for full analysis.

Start with `/archie-scan` for immediate value, then run `/archie-deep-scan` when ready for the full baseline.
